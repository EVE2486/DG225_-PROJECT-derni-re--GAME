# [dernière] — Core Loop & Gameplay [682110135, 682110155]

## Core Loop

```mermaid
flowchart LR
    A[เริ่มต้น] --> B[ตรวจ NPC แต่ละคนว่าผิดปกติหรือไม่ ภายในระยะเวลาที่กำหนด]
    B --> C[ข้อมูลถูกต้อง]
    C --> D[คัดคนเข้าโรงพยาบาล]
    D --> F
    B --> G[ข้อมูลถูกจับได้ว่าปลอมแปลง]
    G --> E[เผาคน]
    E --> H[คนวิกลจริตจริงหรือไม่?]
    H --> I[จริง]
    I --> F
    H --> J[เท็จ]
    J --> K[จบเกม]
    K --> L[GAME OVER]
    F[คะแนนเพิ่ม / Consequence] --> B
```
